# VaultSpace

Open-source, self-hosted secure Virtual Data Room (VDR) platform.

## Overview

VaultSpace provides enterprise-grade document security and collaboration for M&A due diligence, fundraising, board communications, and other sensitive business transactions.

### Key Features (MVP)

- **Multi-tenant architecture** with complete tenant isolation
- **Granular permissions** at room, folder, and document levels
- **Secure document viewing** with watermarking and download controls
- **Comprehensive audit logging** for compliance
- **Self-hosted deployment** via Docker Compose

## Tech Stack

- **Framework:** Next.js 14+ (App Router)
- **Language:** TypeScript
- **Database:** PostgreSQL 15+ with Prisma ORM
- **Queue:** Redis + BullMQ
- **UI:** React 18+ with TailwindCSS
