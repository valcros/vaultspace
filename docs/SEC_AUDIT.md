## Verification Posture

| Status       | Meaning                                                                                                            |
| ------------ | ------------------------------------------------------------------------------------------------------------------ |
| `VERIFIED`   | Evidence exists in the codebase or live system. Test or observation reproducible.                                  |
| `PARTIAL`    | Verified at one layer (e.g. unit) but not yet at all layers (e.g. integration / live).                             |
| `STRUCTURAL` | Enforced by code structure such that the failure mode cannot be expressed (e.g. no API surface for the operation). |
| `DEFERRED`   | Spec applies to a feature that is V1 / out-of-MVP scope.                                                           |
